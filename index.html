About

SkyCortex OS is an evolving digital learning and creative system designed to explore web development, interactive design, and generative technologies.

The project functions as a developing “digital school,” where experimental ideas, applications, and learning tools are continuously built and refined.

Within this system, I work as a Full Stack Java Developer building web applications, CRUD systems, and API-driven platforms using Java, Spring Boot, React, JavaScript, HTML, CSS, and PostgreSQL.
