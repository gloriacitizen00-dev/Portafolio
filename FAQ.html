Frequently Asked Questions

Everything you need to know about SkyCortex OS, how to navigate the portfolio, discover its interactive experiences, and explore every creative project.

What is SkyCortex OS?

SkyCortex OS is a creative digital laboratory that combines technology, storytelling, illustration, UX design, programming, and interactive experiences into one evolving portfolio where ideas become real projects.

What can I explore here?

SkyCortex OS is organized into several creative sections:

  • Play – Interactive tarot experiences and creative games.
  • Read – Original manga and storytelling projects.
  • Build – Programming experiments, interactive applications, and creative coding.
  • Create – Handmade macramé and artistic creations.
  • What's New – The latest projects and recent updates.
  • About – Learn more about the creator and the story behind SkyCortex OS.
  • Let's Work Together – Contact information and collaboration opportunities.

What is the Tarot Game?

The Tarot Game is an original reading experience inspired by the Japanese Maneki-neko lucky cat. It is currently available as a beta version while new features, tools, and reading systems continue to be developed. It is a browser-based game that can be played directly online without downloading or installing anything. Simply open the experience and start exploring the readings, cards, and interactive features.

How do I read the manga?

Every illustration on the homepage can be clicked or tapped to view it in full size. To explore the complete story, simply select Read the Original Manga.

What is Build With Code?

Build With Code is a section dedicated to programming projects, coding experiments, visual effects, data visualizations, UX concepts, and interactive applications created inside SkyCortex OS.

This space allows visitors to explore how technology and creativity can work together. It can also serve as inspiration for learning, practicing programming, creating personal projects, or discovering new ideas.

What is Create Handmade Art?

Create Handmade Art is a virtual catalog featuring original macramé creations handcrafted in Costa Rica.

Each piece is created using traditional textile techniques combined with artistic inspiration from nature, celebrating the beauty, delicacy, and uniqueness of handmade materials.

What can I find in What's New?

What's New is the section where you can discover the latest updates, recently added projects, and new experiences inside SkyCortex OS.

Each project includes a direct button so you can quickly explore it without needing to search through different areas of the portfolio.

Why do some buttons take me directly to a tool?

Some interactive tools are part of larger projects but can also be accessed individually.

These shortcuts were created to improve the user experience and make navigation easier by allowing visitors to access specific experiences directly.

Who created SkyCortex OS?

SkyCortex OS was created and developed by Gloria Martinez in Costa Rica.

The project combines front-end development, UX design, illustration, storytelling, creative coding, and digital experimentation into one continuously evolving platform.

How can I contact you?

The Let's Connect section contains the available ways to get in touch for collaborations, freelance opportunities, creative projects, or professional inquiries.

Can I use the artwork or code?

Unless otherwise stated, all illustrations, original stories, concepts, designs, graphics, branding, and creative content featured on SkyCortex OS belong to the creator.

Some coding experiments may be shared for educational purposes, but they remain part of SkyCortex OS unless explicitly released under another license.

Where was SkyCortex OS created?

SkyCortex OS is an independent creative project proudly created and developed in Costa Rica.

Is SkyCortex OS finished?

Not at all.

SkyCortex OS is a living project that continues to grow with new applications, illustrations, experiments, stories, UX improvements, and creative ideas.

Every visit may reveal something new.
